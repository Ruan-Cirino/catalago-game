Sobre o Catalogo Gamer

O Catalogo Gamer e um sistema web simples desenvolvido com HTML, CSS e JavaScript puro. Ele consome a FreeToGame API para listar jogos gratuitos, apresentar detalhes e permitir favoritos usando armazenamento local.

Tecnologias

HTML, CSS e JavaScript, sem frameworks, bibliotecas externas ou banco de dados.

API

Os dados de jogos, imagens, generos, plataformas e descricoes sao carregados diretamente da FreeToGame API.

Objetivo

Demonstrar consumo de API, manipulacao do DOM, responsividade, navegacao entre paginas e uso de localStorage.
